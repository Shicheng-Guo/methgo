MethGo

a comprehensive tool for analyzing whole genome bisulfite sequencing data

Dependencies

Please install Python 2.7, samtools, and bedtools. MethGo can be run on UNIX/Linux only.

Install MethGo

    $ mkdir ~/.local
    $ cd ~/.local
    $ git clone https://github.com/wwliao/methgo.git
    $ chmod +x methgo/methgo
    $ echo 'export PATH=$PATH:~/.local/methgo' >> ~/.bashrc
    $ source ~/.bashrc

Set up environment with a sample input file

  1. Download our sample input file

    $ cd ~
$ curl -O http://paoyang.ipmb.sinica.edu.tw/~gattaca/methgo_demo.tar.gz
$ tar xvfz methgo_demo.tar.gz
$ cd methgo_demo

  2. Install and activate the virtual environment (10 mins)

    $ curl -O https://pypi.python.org/packages/source/v/virtualenv/virtualenv-12.1.1.tar.gz
$ tar xvfz virtualenv-12.1.1.tar.gz
$ python virtualenv-12.1.1/virtualenv.py --no-site-packages --python=python2.7 .venv
$ source .venv/bin/activate
$ pip install -r ~/.local/methgo/requirements/base.txt
$ pip install -r ~/.local/methgo/requirements/addition.txt

    Our virtual environment includes the following packages:

    • NumPy
    • matplotlib
    • pandas
    • pysam == 0.8.0
    • biopython
    • pyfasta
    • Cython
    • pybedtools

Tutorial

  1. Change to data folder

    $ cd ~/methgo_demo/data

  2. Run MET module (5 mins)

    $ methgo met genes.gtf genome.fa demo.CGmap

    Output figures:

    • Global methylation levels (demo.bulk.mean.png)
    • Methylation level distribution (demo.bulk.hist.png)
    • Methylation levels in different genomic regions (demo.feature.CG.png, demo.feature.CHG.png, demo.feature.CHH.png)
    • Global methylation level of all chromosomes (demo.genomewide.png)

    Output files:

    • Methylation levels of each gene (demo.feature.CG.txt, demo.feature.CHG.txt, demo.feature.CHH.txt)

  3. Run TXN module (5 mins)

    $ methgo txn -t ~/.local/methgo/scripts/txn/tair10_txn -l ATHB1_binding_site_motif,CCA1_binding_site_motif -c demo.CGmap

    Output figures:

    • Methylation level around transcription binding site (demo.CG.txn.png, demo.CHG.txn.png, demo.CG.txn.png)

  4. Run CNV module (5 mins)

    $ methgo cnv genome.fa.fai demo.sorted.bam

    Output figure:

    • CNV on all chromosomes (demo.sorted.cov.png)

  5. Run SNP module (5 mins)

    $ methgo snp -g genome.fa demo.sorted.bam

    Output files:

    • homozygous SNPs (demo.sorted.homozygous.txt)
    • heterozygous SNPs (demo.sorted.heterozygous.txt)
